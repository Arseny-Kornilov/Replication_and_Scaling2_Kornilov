### Задание 1.
Опишите основные преимущества использования масштабирования методами:
#### - активный master-сервер и пассивный репликационный slave-сервер;
ОТВЕТ: Отказоустойчивость, поскольку при падении Master можно быстро переключиться на Slave, низкая нагрузка при выборке и чтении данных, поскольку slave служит только для чтения, возможность резервного копирования со slave без нагрузки на master, простая настройка и управление.
#### - master-сервер и несколько slave-серверов;
ОТВЕТ: Более высокая отказоустойчивость за счет распределения нагрузки между несколькими репликами, возможность создания реплик под разные типы запросов, высокий уровень безопасности.
